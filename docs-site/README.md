# Docs Site

This Docusaurus site is the documentation hub for **LLM Project Mapper**.

## Source of truth

- `docs-site/docs/` is generated by `../scripts/sync-docs-site.mjs`.
- Canonical content stays in the repository root docs such as `README.md`, `INSTALL.en.md`, `INIT.en.md`, `docs/`, and `.specs/workflow/CONTRIBUTING.md`.
- Version snapshots under `versioned_docs/` are created from the generated docs when you cut a release doc version.

## Commands

```bash
npm install
npm run build
npm run start
npm run docs:version -- 0.3.0
```

The GitHub Actions workflow at `../.github/workflows/docs-site.yml` deploys this site to GitHub Pages on pushes to `main`.
